# SpaceSiege <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Flying%20Saucer.png" alt="Flying Saucer" width="40" height="40" />
A web-based space invaders game where the player defends it-self from incoming enemy siege.
The player controls a spaceship and must destroy all incoming enemies before they reach the 
bottom of the screen or before running out of lives.


## Game Overview
Space Siege is a fast-paced arcade game where the player pilots a spaceship and shoots enemy invaders descending from space.
The objective is to eliminate all enemies while avoiding being overwhelmed.

* The game demonstrates fundamental concepts of game development including:
* Real-time player input
* Object movement and collision detection
* Game loops and animation
* Score tracking

## How to Play
Controls are simple and easy to learn.
* Left Arrow – Move spaceship left
* Right Arrow – Move spaceship right
* Up Arrow - Move spaceship up
* Down Arrow - Move spaceship down
* Spacebar – Fire laser

Gameplay:
* Enemies descend from the top of the screen.
* The player moves the spaceship horizontally and vertically.
* Lasers can be fired to destroy enemies.
* The game continues until the player eliminates enemies or they reach the player.


## How the Game Was Built
1. The development process included several steps:
2. Setting up a development environment using GitHub Codespaces.
3. Implementing the game structure using HTML and JavaScript.
4. Creating the player spaceship and enemy objects.
5. Implementing movement, shooting mechanics, and collision detection.
6. Adding animations and rendering using HTML Canvas.
7. Testing gameplay mechanics and fixing bugs.
8. Deploying the final game to the cloud.


## Features
* Player controlled spaceship
* Enemy waves descending from space
* Laser shooting mechanics
* Collision detection system
* Real-time gameplay using a game loop
* Simple and responsive controls
