# SpaceSiege <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Flying%20Saucer.png" alt="Flying Saucer" width="40" height="40" />
A web-based space invaders game where the player defends it-self from incoming enemy siege.
The player controls a spaceship and must destroy all incoming enemies before they reach the 
bottom of the screen or before running out of lives.
